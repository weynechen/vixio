# Xiaozhi Web Client

A simple web simulator for testing the Xiaozhi server without physical hardware.

## Prerequisites

1. **Node.js** installed.
2. **Xiaozhi Server** running locally (default port: `8000`).

## Quick Start

1. Go to the web directory:
   ```bash
   cd web/xiaozhi
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Open the link shown in the terminal (usually `http://localhost:5173`).

## Usage

1. Click **Connect** to connect to the server.
2. The client will automatically start **Listening**.
3. Speak into your microphone to chat.
4. Click **Interrupt** to stop the bot while it is speaking.
